# Business Analytics Dashboard

## Setup & Run

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Open Browser**:
    Visit `http://localhost:5173` (or the URL shown in the terminal).

## Features
- **CSV Upload**: Drag & drop or select a file.
- **Dynamic Charts**: Bar, Line, Pie, Area.
- **Configuration**: Select X and Y axes dynamically.
- **Dark Mode**: Premium UI inspired by modern dashboards.
